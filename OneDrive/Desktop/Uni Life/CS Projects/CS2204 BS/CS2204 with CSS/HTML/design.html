Rules usage and its explanation

I used position: relative in lines 76-78 of home.css to move the Location heading a bit

I used animation on heading of every page. The code is in line 81-88 and 5-6 of home.css, it changes the color from orange to white in 4 seconds. I used the same code for every page


I used transition in lines 86 and 94 of apply.css. The transition period is very big so the transition itself could be seen


I used border-radius in line 28 and 38 of about.css and line 150 of apply.css to make round corners


I used embedded style in lines 8-20 of design.html, which adds style to design.css. design.css consists only of rules that every other page has and there are only 2 exclusive rules this page has


Additionally, I added inline style in line 114 of apply.html, since it was only one parameter that needed to be changed. (As I've understood, the points are given for using all three styles)
